Projects and Configuration
##########################

.. image:: _static/logo/pyEDAA.ProjectModel_light.svg
   :height: 80 px
   :target: https://github.com/edaa-org/pyEDAA.ProjectModel

**pyEDAA.ProjectModel** provides a generic vendor and tool independent description of an EDA project.
It reflects aspects like multiple *design variants* of the same project, grouping of source files into *file sets*
or linking *test benches* to components or subsystems in a design. To respect various use cases, files can be attached
with user defined property fields.
