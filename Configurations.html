Projects and Configuration¶

ProjectModel¶

pyEDAA.ProjectModel ➚ provides a generic description of an EDA project, independent of vendor and tools. It reflects aspects like multiple design variants of the same project, grouping of source files into file sets or linking test benches to components or subsystems in a design. To respect various use cases, user defined property fields can be attached to file(sets).
